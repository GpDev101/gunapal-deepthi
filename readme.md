# 💍 Wedding Invitation Website

A modern, interactive South Indian wedding invitation website built with React + Vite and deployed on Vercel.

Designed with elegant animations, smooth transitions, cartoon bride & groom illustrations, event timelines, Google Maps integration, countdown timer, music, and mobile-first responsiveness.

---

# ✨ Features

- 🎉 Elegant landing page with cinematic hero section
- 👰 Cartoon-style bride & groom interactive visuals
- ⏳ Live wedding countdown timer
- 📍 Google Maps integration for venue navigation
- 📅 Event timeline for multiple ceremonies
- 🎵 Optional background music toggle
- 📸 Gallery / memories section
- 💌 Personalized invitation message
- 📱 Fully responsive mobile-first design
- 🌙 Smooth animations and transitions
- 🚀 Vercel-ready deployment setup
